# fayver-app

## Fayver es una To-do App inspirada en aplicaciones como Notion, Trello y Jira. Permite gestionar proyectos, objetivo y tareas de manera intuitiva, sencilla y completamente personalizada.

**[Ver Demo en Vivo](https://fayver-app.vercel.app/)**

Fayver comienza como un simple proyecto de 2º CFGS de Desarrollador de Aplicaciones Web, y escala como una aplicación web completamente funcional, estable y usable por cualquier usuario a su alcance.
Esta aplicación esta realizada con las siguientes tecnologías:

- Back-end: Node.js + Express.
- Front-end: React + TailwindCSS
- Base de datos: MySQL
- Entorno de desarrollo: Visual Studio Code.
- Cliente SQL: MySQL Workbench
- Despliegue Back-end: Render
- Despliegue front-end: Vercel
