# fayver-app

Fayver es una To-do App inspirada en aplicaciones como Notion, Trello y Jira. Permite gestionar proyectos, objetivos y tareas de manera intuitiva, sencilla y completamente personalizada.
Esta idea comienza como un simple proyecto de 2º CFGS de Desarrollo de Aplicaciones Web, y ha escalado hasta convertirse en una aplicación web completamente funcional, estable y usable por cualquier usuario.

**[Ver Demo en Vivo](https://fayver-app.vercel.app/)**

---------

## Tecnologías usadas:

- **Back-end:** Node.js + Express.
- **Front-end:** React + TailwindCSS
- **Base de datos:** MySQL
- **Entorno de desarrollo:** Visual Studio Code.
- **Cliente SQL:** MySQL Workbench
- **Despliegue Back-end:** Render
- **Despliegue front-end:** Vercel

## Características principales:

- **Autenticación segura**: Proceso de autenticación y cifrado seguros, usando JWT para guardar el estado de sesión del usuario y bcrypt para cifrar las contraseñas.
- **Gestión de tablones personalizada:** Crea tablones, listas y tareas completamente personalizadas para organizar tu día a día.
- **Tablero interactivo:** Disfruta de un sistema de Drag & Drop para organizar tus listas y tareas de manera rápida y sencilla. Agrega etiquetas, colores y contenido para diferenciar tus tareas con facilidad.
- **Función de archivado:** No elimines tus tareas tras haberlas realizado, puedes archivarlas para tener constancia de que has sido capaz de realizarlas con éxito.
- **Historial de actividades:** Visualiza mediante una barra lateral desplegable las actividades que has realizado en cualquier tablón en todo momento de manera automática.
- **Diseño responsivo:** Accede a Fayver desde cualquier dispositivo y disfruta de su usabilidad y accesibilidad.
